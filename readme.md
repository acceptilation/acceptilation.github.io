# Notes

My public notes in the form of an MkDocs site

## Python setup

```shell
sudo apt install python3-full;
python3 -m venv ./.venv;
source ./.venv/bin/activate;
pip install --requirement requirements.txt;
```

## Run MkDocs

```shell
mkdocs serve;
```

## JetBrains run configuration

To run site locally:\
`mkdocs-serve`
